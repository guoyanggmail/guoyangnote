## 🤔 MCP 是什么？

MCP (Model Context Protocol) 是 Anthropic 公司于 2024 年底开源的一项技术，简单来说，它就是 AI 大模型的标准化工具箱。MCP (Model Context Protocol) 是 Anthropic 公司在 2024 年底开源发布的一项技术，简单来说，它就是 AI 大模型的标准化工具箱。大模型可以利用这些工具与外界互动，获取信息并完成具体任务。

在日常工作学习中，我们经常需要与浏览器、文件、数据库、代码仓库等外部工具打交道。传统方式需要手动截图或复制文本，然后粘贴到 AI 窗口进行对话。而 MCP 通过标准化的协议自动化了这一步骤。

MCP 服务作为 AI 与外部工具的中间层，代替人类访问并操作外部工具。每个 MCP 服务 (或 MCP Server) 都专精于一类工作，例如有的负责读写浏览器，有的负责读写本地文件，有的操作 GitHub 仓库等等。

MCP Server 通常是运行在本地的一段 Node.js 或 Python 程序。大模型通过操作系统的标准输入通道 (STDIO) 调用某个 MCP Server，消息格式类似 JSON。大模型通过操作系统的标准输入通道 (STDIO) 调用某个 MCP Server，消息格式类似 JSON。MCP Server 接收到请求后，通过自己的代码功能或使用 API 请求访问外部工具完成任务。

看到这里，你可能会觉得 MCP 协议与 Function Call 非常相似。MCP 的最大优点在于整合了之前各家大模型不同的 Function 标准，整合成了一个统一的标准协议。而且不仅仅是 Anthropic 的 Claude，市面上几乎所有的大模型都可以接入 MCP。
