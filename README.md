## 使用说明

本知识库基于Obsidian管理，建议使用Obsidian打开以获得最佳体验。

## 提交规范

本项目遵循[Git提交规范](./Normal/Git提交规范.md)，提交格式为：

```
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

常用的提交类型包括：
- **feat**: 新增功能
- **fix**: 修复Bug
- **docs**: 文档变更
- **style**: 代码格式调整
- **refactor**: 代码重构
- **chore**: 其他变更
