# Mac工具推荐

## 1. BetterDisplay

BetterDisplay是一款强大的显示器管理工具，可以帮助您更好地管理和优化Mac的显示设置。

### 主要功能
- 支持多显示器管理
- 自定义显示器分辨率
- 显示器亮度、对比度调节
- 显示器旋转和镜像
- 显示器排列方式调整
- 支持HiDPI模式

### 安装方法
1. 访问官方网站：[https://betterdisplay.pro/](https://betterdisplay.pro/)
2. 下载安装包
3. 双击安装包进行安装
4. 首次运行时需要授予系统权限

### 使用教程
1. 打开BetterDisplay，在菜单栏可以看到图标
2. 点击图标可以快速访问常用功能
3. 在偏好设置中可以自定义快捷键
4. 支持为不同显示器设置不同的配置方案

## 2. OrbStack

OrbStack是一个现代化的开发环境管理工具，特别适合Docker和虚拟机管理。

### 主要功能
- Docker容器管理
- Linux虚拟机管理
- 开发环境快速部署
- 网络管理
- 资源监控
- 团队协作支持

### 安装方法
1. 访问官方网站：[https://orbstack.dev/](https://orbstack.dev/)
2. 下载Mac版本安装包
3. 安装完成后启动应用

### 使用教程
1. 首次启动会引导您完成初始化设置
2. 可以快速创建Docker容器或Linux虚拟机
3. 支持通过命令行或图形界面管理
4. 可以设置自动启动和资源限制

## 3. Mac Mouse Fix

Mac Mouse Fix是一款优化Mac鼠标体验的工具，特别适合使用第三方鼠标的用户。

### 主要功能
- 自定义鼠标按键
- 平滑滚动
- 智能加速
- 手势支持
- 多设备管理
- 配置文件导入导出

### 安装方法
1. 访问官方网站：[https://macmousefix.com/](https://macmousefix.com/)
2. 下载安装包
3. 安装并授予系统权限

### 使用教程
1. 打开Mac Mouse Fix
2. 在偏好设置中配置鼠标按键
3. 可以设置不同应用的特定配置
4. 支持创建多个配置文件

## 4. Switch Hosts

Switch Hosts是一个简单易用的hosts文件管理工具，方便开发者进行本地开发环境配置。

### 主要功能
- hosts文件管理
- 多环境配置
- 快速切换
- 远程hosts同步
- 规则导入导出
- 分组管理

### 安装方法
1. 访问GitHub：[https://github.com/oldj/SwitchHosts](https://github.com/oldj/SwitchHosts)
2. 下载最新版本
3. 安装应用

### 使用教程
1. 首次运行需要输入管理员密码
2. 创建新的hosts规则
3. 可以设置规则组
4. 支持快速启用/禁用规则
5. 可以导入导出配置

## 总结

这些工具都是Mac上非常实用的生产力工具，能够显著提升您的工作效率：
- BetterDisplay让显示器管理更加便捷
- OrbStack简化了开发环境的管理
- Mac Mouse Fix优化了鼠标使用体验
- Switch Hosts方便了开发环境的配置

建议根据个人需求选择合适的工具进行安装使用。如果您有其他好用的Mac工具推荐，欢迎补充。
